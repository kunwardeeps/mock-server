# Mock-server
- Spring boot application to mock API requests quickly. 
- All mock requests are persisted in MongoDB. 
- Can be used to mock SOAP and REST APIs

Installation Notes:
1. Install MongoDB
2. Run the app using spring boot or create an uber jar to deploy
